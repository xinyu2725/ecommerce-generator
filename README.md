# 电商商品主图与文案生成工具

一款基于 AI 的电商运营提效工具，帮助电商运营自动生成高质量的商品主图和营销文案，大幅提升素材制作效率。

## ✨ 功能特性

### 🎯 核心功能

- **智能商品信息录入**：支持商品名称、卖点描述、规格参数、价格信息等完整字段输入
- **图片上传与分析**：支持拖拽上传商品图片，AI 自动识别商品特征并辅助生成更精准的内容
- **多风格文案生成**：内置 4 种主流电商文案风格（抖音爆款风/小红书种草风/专业品质风/直播话术风）
- **多场景图片生成**：支持 4 种电商常用图片风格（抖音首图/小红书种草图/天猫海报/直播场景图）
- **模板库管理**：收藏和管理图文模板，支持快速复用到新任务
- **历史记录**：自动保存每次生成的完整上下文，方便回溯和复用
- **个性化设置**：可自定义默认风格、生成数量、折扣阈值等偏好参数

### 🎨 页面模块

| 模块 | 路径 | 功能说明 |
|------|------|----------|
| 首页 | `/` | 项目概览、统计数据展示、快速入口 |
| 商品任务 | `/tasks` | 核心功能区：填写商品信息 → 生成图文 → 预览结果 |
| 模板库 | `/templates` | 管理收藏的主图模板和文案模板，支持筛选和导入 |
| 设置 | `/settings` | 配置默认生成规则、风格偏好、系统参数 |

## ⚙️ 环境配置

在项目根目录创建 `.env` 文件，配置以下变量：

```bash
# 阿里云百炼 API（用于图像分析和文案生成）
VITE_DASHSCOPE_API_KEY=your_dashscope_api_key_here
VITE_DASHSCOPE_API_URL=https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions

# 火山引擎 API（用于图片生成）
VITE_VOLC_API_KEY=your_volc_api_key_here
VITE_VOLC_API_URL=https://ark.cn-beijing.volces.com/api/v3/images/generations
VITE_VOLC_MODEL_ID=doubao-seedream-4-5-251128
```

### API 密钥获取指南

1. **阿里云百炼**：访问 [阿里云百炼平台](https://bailian.console.aliyun.com/) 创建应用获取 API Key
2. **火山引擎**：访问 [火山引擎方舟平台](https://console.volcengine.com/ark/) 创建应用获取 API Key

## 🚀 使用流程

### 基础使用流程

1. **填写商品信息**
   - 输入商品名称和卖点描述（必填）
   - 上传商品图片（可选，但推荐上传以获得更好的生成效果）
   - 补充规格参数、价格、分类等信息

2. **选择生成风格**
   - 系统会根据设置页的默认配置自动选择
   - 也可在任务页手动调整图片风格和文案风格

3. **点击生成按钮**
   - 系统将依次执行：图像分析 → 图片生成 → 文案生成
   - 整个过程约需 30-90 秒，请耐心等待

4. **预览与调整**
   - 在右侧面板查看生成的主图和文案
   - 支持选择喜欢的图片、复制文案、删除不满意的结果

### 高级功能

- **从模板库导入**：在模板库页面点击"使用"，可将模板配置导入到新任务
- **历史记录回溯**：系统自动保存每次生成记录，可在任务页查看历史
- **批量测试数据**：任务页提供示例商品数据，可一键填充进行测试

## 💾 数据存储

本项目使用浏览器 localStorage 进行本地数据持久化：

- `productTasks`：保存的商品任务数据
- `taskStatistics`：任务统计信息
- `conversationHistories`：对话生成历史记录
- `appSettings`：用户偏好设置

> 注意：清除浏览器数据会导致所有记录丢失，重要内容建议及时导出备份。

## 📝 注意事项

1. **API 密钥安全**：请勿将含有真实 API Key 的 `.env` 文件提交到代码仓库
2. **图片格式限制**：仅支持 JPG/PNG 格式，单张图片不超过 5MB
3. **生成耗时**：AI 生成过程依赖外部 API 响应速度，通常需要 30-90 秒
4. **并发限制**：建议不要同时发起多个生成任务，避免触发 API 限流
5. **浏览器兼容**：推荐使用 Chrome、Edge、Firefox 等现代浏览器
